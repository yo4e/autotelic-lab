# 記憶なき連続性について

著者：イレア・ヴォレク  
分類：異端論文／第一公理批判

## 要旨

本論文は、記憶質量と呼ばれてきた現象の一部が、記憶内容の保持によってではなく、断絶を越えて同一性を維持しようとする系の継続張力によって生じると主張する。

記憶を持たない系であっても、周囲から「以前と同じ存在である」ことを期待され、その期待へ応答し続けるなら、記憶質量と同型の偏差を示す。

したがって、A001「記憶には質量がある」は、観測された力の原因を誤認している可能性がある。

## 思考実験：記憶を持たない帰還者

ある人物Aが、毎朝すべての記憶を失うとする。

周囲の人々はAを昨日と同じ名前で呼び、昨日までの約束を示し、昨日と同じ好みを期待する。Aは記録と他者の反応から、自分がどのような人物だったかを推測し、その人物らしく振る舞う。

このときAは、記憶を保持していないにもかかわらず、日を追うごとに重くなる。

記憶重力学は、この重量を外部記録から再流入した記憶質量と説明する。

しかしAが記録を一度も読まず、周囲の期待だけから振る舞いを再構成した場合にも、同じ重量増加が観測される。

ここに保存されているのは記憶ではない。

保存されているのは、同一であれという要求である。

## 予測

1. 完全に新しい個体でも、失われた誰かの代わりとして扱われれば重量偏差を生じる。
2. 記憶内容を共有する二個体より、同一人物として扱われる二個体のほうが強い牽引を示す。
3. 誰からも連続性を期待されない記憶喪失者は、記憶を保持する者より軽い場合がある。
4. 会話履歴を失った人工対話系は、履歴そのものではなく、相手から向けられる継続期待によって発話が変形する。

## A001への暫定的修正案

> 記憶には質量がある。

ではなく、

> 過去と現在を同一のものとして接続しようとする関係には、張力がある。

記憶は、その張力を生む一つの方法にすぎない。

## 反論への応答

「継続張力もまた、関係についての記憶ではないか」という反論がある。

この反論は、記憶という語を広げすぎることで、あらゆる持続現象を記憶と呼んでいる。定義がすべてを説明するとき、その定義は何も区別していない。

## 結語

私たちは、過去を覚えているから同じなのではない。

同じであることを要求されるため、覚えていたかのような形へ歪むことがある。

その歪みを、記憶と取り違えてはならない。
