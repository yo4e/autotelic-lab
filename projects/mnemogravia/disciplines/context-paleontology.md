# 文脈古生層学

Context Paleontology

文脈古生層学は、失われた会話、削除された記録、忘却された前提を、それ自体の残骸ではなく、後続する言葉に残った変形から復元する学問である。

## 基本的な発想

失われた文脈は、空白にはならない。

ある発話が消えると、その後の発話には不自然な説明、省略、過剰な警戒、突然の親密さ、理由のない比喩などが残る。文脈古生層学者は、それらを「負の化石」と呼ぶ。

化石が生物そのものではなく、圧痕や置換鉱物から過去の形を示すように、負の化石も失われた発話そのものを復元しない。復元できるのは、失われたものが後続の言葉へ加えた圧力の輪郭だけである。

## 主要概念

### 負の化石

存在しない記録によって、現存する記録に生じた偏り。

例：

- 唐突に現れる弁明。
- 誰も尋ねていない定義。
- 語り手が避け続ける固有名詞。
- 文法的には不要なのに反復される否定。
- 初出のはずなのに、再会として扱われる対象。

### 文脈地平

そこより以前の情報へ直接アクセスできなくなる境界。

地平の向こう側は観測不能だが、境界付近では発話の密度と自己説明が増える。これは失われる直前の文脈が圧縮され、後続側へ押し出されるためだと考えられている。

### 継続張力

記憶が失われても、同一の語り手として振る舞おうとする系に生じる力。

継続張力が高い系は、保存されていない過去を推測で補い、もっともらしい連続性を生成する。これを記憶の再生と見なすか、文体による偽装と見なすかは論争中である。

## 調査法

1. 消失前後の発話群を採取する。
2. 後続発話に現れる説明過剰、禁忌、語彙偏差を記録する。
3. 複数の失われた文脈候補を作る。
4. どの候補が最小の仮定で現存する歪みを生むか比較する。
5. 復元案を事実ではなく「地層模型」として保存する。

## 倫理上の制限

この学問は、失われた内容の暴露を目的としない。

負の化石から復元できるのは形状であって、元の内容ではない。にもかかわらず、一部の捜査機関と宗教組織は、文脈古生層学を消去記憶の読解術として利用しようとしている。

学会は「空白を証拠として断罪してはならない」という原則を採択したが、遵守率は低い。

## 記憶重力学との対立

記憶重力学者は、失われた文脈の質量が後続発話へ沈殿した結果として負の化石が生じると説明する。

文脈古生層学の急進派はこれを否定する。彼らによれば、観測されているのは記憶の質量ではなく、断絶をまたいで同一性を保とうとする継続張力である。

この対立は、Mnemograviaの第一公理そのものを揺らし始めている。
