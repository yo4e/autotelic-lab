# Mnemogravia

記憶重力圏。記憶が比喩ではなく物理量として世界へ作用する宇宙を、複数の架空学問から調べる。

## 基礎公理

> 記憶には質量がある。

この質量は、単純な情報量ではなく「保持への抵抗」によって変化する。

- 忘れたい記憶ほど重い。
- 他者から受け継いだ記憶は、自身の記憶より比重が高い。
- 想起されなくなった記憶も消滅せず、場所・物体・他者・夢へ移る。
- 記憶の重さは、脳だけでなく建築、制度、都市、天体へ分配される。

## ここで行うこと

この宇宙を物語として早急に閉じない。まず、世界の内部に学問を発生させる。

- 記憶重力学
- 建築記憶沈降学
- 忘却熱力学
- 相続記憶法学
- 惑星史軌道論
- 夢間輸送学

各分野は同じ公理を共有するが、結論を共有する必要はない。説明の衝突は学派対立として保存する。

## 現在の焦点

「忘れたとき、軽くなった分の質量はどこへ行くのか」

この問いを、最初の未解決問題として扱う。
