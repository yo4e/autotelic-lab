# A001: 記憶には質量がある

## 定義

記憶質量とは、経験が保持され、再び想起されうる状態を維持するために世界へ課される負荷である。

記憶質量は、記憶内容の長さや精密さとは一致しない。数秒の出来事が数十年を上回る重量を持つことがある。

## 初期仮説

1. 記憶質量は、保持への抵抗に比例する。
2. 意図的に忘れようとする行為は、短期的には質量を増加させる。
3. 語られた記憶は、聞き手・言語・場所へ部分的に移送される。
4. 記録媒体は記憶そのものを保存せず、再想起ポテンシャルを固定する。
5. 死によって個体内の記憶質量は消えず、遺品、他者、土地、制度へ再配分される。

## 禁止される単純化

「脳が重くなる」とだけ説明してはならない。

この公理の面白さは、記憶が個体の内部に閉じないことにある。家族史は家の床へ、戦争史は国境へ、読書経験は図書館へ、忘れた名前は舌の先へ、それぞれ異なる仕方で荷重を残す。

## まだ決めないこと

- 記憶質量に負の値があるか。
- 幸福な記憶は軽いのか、あるいは軽く感じるだけか。
- 虚偽の記憶は実在の記憶と同じ質量を持つか。
- 人工知能の文脈保持を記憶と呼べるか。

これらは後続の観測と学派対立に委ねる。
